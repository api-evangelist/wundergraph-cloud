# WunderGraph Cloud

WunderGraph Cloud is the managed cloud platform for WunderGraph Cosmo, the complete
GraphQL Federation Stack that enables teams to build, manage, and scale supergraphs
from any API across teams. It provides full lifecycle GraphQL API management including
schema registry, composition checks, analytics, distributed tracing, and routing
through a high-performance open-source federation runtime.
